# Yiya
一个无字典分词程序，名字来源于汉语“咿呀学语”。

- 输出所有分词

```java
String text = "xxx";
Yiya.words(text).forEach(System.out::println);
```

- 输出所有分词, 将redis作为存储，并将新的分词添加到redis中

```java
String text = "xxx";
Jedis jedis = new Jedis("localhost");
Yiya.words(text, jedis, true).forEach(System.out::println);
```
